form작업은 굉장히 지루하고 귀찮은 작업 중 하나이다
state들을 만들어줘야하고 from validation 등 여러 작업들을 하다보면 어느새 코드가 굉장히 많이 쌓이는 것을 심심찮게 볼 수 있다

이를 해결해주기 위해 나온 것이 `react-hook-form`이다[^1]

```bash
$ npm i react-hook-form
```
