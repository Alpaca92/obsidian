# 1-1: 컴포넌트에 대하여
## 컴포넌트는 무엇일까요?
### 웹 컴포넌트
웹 컴포넌트는 그 기능을 나머지 코드로부터 캡슐화하여 재사용 가능한 커스텀 엘리먼트를 생성하고 웹 앱에서 활용할 수 있도록 해주는 다양한 기술들의 모음[^2]

```ad-quote
title: 웹 컴포넌트가 표준인 이유는 별도의 설치가 필요없기 때문이다
```
